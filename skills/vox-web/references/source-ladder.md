# Web source ladder (most → least fetchable)

Prefer earlier entries. Discover unguessable aggregator slugs via `WebSearch '<name> <site>'`
first, THEN fetch.

1. Official non-JS sites; Slice (slicelife.com); MenuPages; allmenus.com; Chowbus (pos.chowbus.com).
2. Review/critic: The Infatuation, Restaurantji, RestaurantGuru, Tripadvisor listing, Wanderlog,
   critic Substacks, Michelin description text.
3. WebSearch result-page TITLES and SNIPPETS — first-class data (carry prices/ratings/review
   counts even when the page itself 403s).

Conflict resolution: prefer two agreeing CURRENT sources over one old one; official > stale
aggregator; delivery-platform prices are a CEILING (~15–30% markup) — label as such. Never average.
